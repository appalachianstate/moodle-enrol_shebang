Logging Settings

These settings affect message and process logging.

  • Process Log: Write only process errors - when checked, no entries will be made when the message import is successful.
  • Message Log: Write incoming LMB messages - when checked, each message received from the LMB will logged to a file in the enrol_shebang sub-directory of the moodledata directory.
  • Message Log: Suspend Locking Log - no serialization. The message log file will not be locked exclusively as each LMB message import worker process handles its POST.
